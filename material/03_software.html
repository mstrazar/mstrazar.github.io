The list of open source software developed and contributed to.

CAPTAn: Context-Aware predictor of T-cell antigens. Prediction of peptide binders to MHC-II molecules based on sequences of protein antigens Author. | GitLab repository.

openTSNE: Modular implementation of t-Distributed Stochasitc Neighbor Embedding (t-SNE). Python. Method co-author and contributor. | GitHub repository

MKLaren: A Multiple kernel learning library. Python. Author. | GitHub repository

csDEX: Condition-specific differential exon expression based on generalized linear models. R. Author. | GitHub repository

iONMF: Integrative orthogonal non-negative matrix factorization. Python. Author. | GitHub repository

Switch-IT: Molecular models of a bistable switch based on designed DNA binding elements. Matlab, C, Python. Author. | GitHub repository

Orange Data Mining: General purpose data mining, machine learning and visualization tools. Python, C++. Contributor. GitHub repository | Project page

Single cell Orange (scOrange): Orange Extension for analysis on single-cell RNA sequecing data. Python, C++. Contributor. | GitHub repository | Project page
